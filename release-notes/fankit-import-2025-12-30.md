How to update
-------------
Use the provided import scripts to refresh or add assets from a local Fankit source:

- Unix:
  ```bash
  ./scripts/import-fankit.sh /path/to/Fankit_Source
  ```
- Windows PowerShell:
  ```powershell
  ./scripts/import-fankit.ps1 -SourcePath "C:\Path\To\Fankit"
  ```
